\doxysubsection{Detailed Description}
A Null Object implementation of \doxylink{classcompression_1_1ICompressor}{ICompressor}. 

This class implements the \doxylink{classcompression_1_1ICompressor}{ICompressor} interface but performs no actual compression or decompression. It simply returns the input data unchanged.
