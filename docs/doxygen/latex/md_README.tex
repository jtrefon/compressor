A high-\/performance file compression library implemented in C++ that provides multiple compression algorithms with a focus on achieving excellent compression ratios for files at rest.\hypertarget{md_README_autotoc_md1}{}\doxysection{\texorpdfstring{Features}{Features}}\label{md_README_autotoc_md1}

\begin{DoxyItemize}
\item Multiple compression algorithms\+:
\begin{DoxyItemize}
\item {\bfseries{Null Compressor}}\+: Reference implementation with no compression
\item {\bfseries{RLE (Run-\/\+Length Encoding)}}\+: Simple compression for data with repeated patterns
\item {\bfseries{Huffman Coding}}\+: Statistical compression using variable-\/length codes
\item {\bfseries{LZ77}}\+: Dictionary-\/based compression using sliding window technique
\item {\bfseries{Deflate}}\+: Combined LZ77 and Huffman coding (similar to gzip/zlib)
\end{DoxyItemize}
\item Optimized implementations\+:
\begin{DoxyItemize}
\item Fast hash-\/based string matching for LZ77
\item Efficient bit-\/level encoding and decoding
\item Robust error handling for corrupted data
\end{DoxyItemize}
